4단계 스코어링 기준

각 조건을 충족할수록 점수가 올라가고, 4점 만점이면 Strong Buy 신호가 됩니다.

1
수익성 — ROE > 10%
자기자본이익률이 10%를 초과하면 지속적으로 수익을 창출하는 기업으로 평가
2
재무 건전성 — D/E < 100
부채비율이 100 미만이면 재무 구조가 안정적이라고 판단
3
밸류에이션 — 현재 PER < 5년 평균 PER
현재 PER이 5년 평균보다 낮으면 역사적으로 저평가된 구간으로 판단
4
기술적 추세 — 현재가 > MA120
120일 이동평균선 위에서 거래되면 장기 상승 추세가 유지되고 있다고 판단

포지션 사이징 공식
최대 허용 손실 총자산 × 2% → 기본 $2,000
주당 손실액 현재가 × 10% → 10% 손절 기준

기본 실행

python 
from stock_analyzer import analyze_long_term_stocks, print_report

# 미국 주식
tickers = ["AAPL", "MSFT", "GOOGL", "NVDA"]
df = analyze_long_term_stocks(tickers, total_capital=100_000)
print_report(df)

# 한국 주식
kr = ["005930.KS", "000660.KS"]
df_kr = analyze_long_term_stocks(kr)

# CSV 저장
df.to_csv("result.csv", index=False)
