Scoring Logic

4단계 스코어링

1
수익성 — ROE > 10%
자기자본이익률이 10%를 초과할 경우 우수한 수익 창출 기업으로 평가
2
재무 건전성 — D/E < 100
부채비율이 100 미만일 경우 안정적인 재무 구조로 판단
3
밸류에이션 — 현재 PER < 5년 평균 PER
현재 주가수익비율이 5년 평균보다 낮으면 역사적 저평가 구간으로 간주
4
기술적 추세 — 현재가 > MA120
120일 이동평균선 위에서 거래될 경우 장기 상승 추세 유지로 판단

포지션 사이징 공식

최대 허용 손실 = 총자산 × 2% → 기본 $2,000 주당 손실액 = 현재가 × 10% → 손절 기준 권장 매수 수량 = floor(최대손실 / 주당손실액)
Usage

사용법

설치

bash 
pip install yfinance pandas numpy

기본 실행

python 
from stock_analyzer import analyze_long_term_stocks, print_report

# 미국 주식
tickers = ["AAPL", "MSFT", "GOOGL", "NVDA"]
df = analyze_long_term_stocks(tickers, total_capital=100_000)
print_report(df)

# 한국 주식 (삼성전자, SK하이닉스)
kr_tickers = ["005930.KS", "000660.KS"]
df_kr = analyze_long_term_stocks(kr_tickers, total_capital=100_000)

# CSV 저장
df.to_csv("stock_analysis.csv", index=False)

설정값 커스터마이징

stock_analyzer.py 
TOTAL_CAPITAL   = 100_000  # 총 투자 자산 (USD)
MAX_LOSS_PCT    = 0.02     # 포지션당 최대 손실 비율 (2%)
STOP_LOSS_PCT   = 0.10     # 손절 기준 (10%)
ROE_THRESHOLD   = 0.10     # ROE 기준선 (10%)
DE_THRESHOLD    = 100      # 부채비율 기준선
MA_TREND_WINDOW = 120      # 추세 판단 이동평균 기간 (일)
